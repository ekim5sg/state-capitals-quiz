# U.S. State Capitals Quiz – Python & Rust Yew WASM

This repository contains three versions of a U.S. state capitals learning app:

### 1. Python Tkinter (Map Version)
`python-map-tkinter/`  
Kids answer capitals and the map reveals labels in the correct locations.

### 2. Python Tkinter (Simple Version)
`python-quiz-tkinter/`  
Green ✔ / red ✘ feedback inside the app, no repeated states.

### 3. Rust Yew + WebAssembly Version
`state-capitals-quiz-yew/`  
The fast, modern browser version.  
Live demo: https://www.webhtml5.info/state-capitals-quiz-yew/
